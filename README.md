# 💳 Credit Card Fraud Detection System

## 📌 Project Overview

This project is an end-to-end **Credit Card Fraud Detection System** built using Machine Learning. It detects whether a transaction is **fraudulent or legitimate** based on user behavior and transaction patterns.

The system includes:

* 🔙 **Backend**: Built with Flask for model integration and API handling
* 🎨 **Frontend**: User-friendly interface for input and prediction
* 🤖 **Machine Learning Models**: Logistic Regression & Random Forest

---

## 🚀 Features

* Real-time fraud prediction
* User-friendly web interface
* High accuracy ML models
* Feature-based detection (velocity, device trust, location mismatch)
* Scalable backend using Flask

---

## 🧠 Machine Learning Models Used

* **Logistic Regression** → For baseline prediction
* **Random Forest Classifier** → For improved accuracy and robustness

---

## 📊 Key Features (Input Variables)

* Transaction Amount
* Transaction Time
* Velocity (Last 24h)
* Device Trust Score
* Location Mismatch
* Other behavioral features

---

## 🏗️ Tech Stack

* **Frontend**: HTML, CSS
* **Backend**: Flask (Python)
* **ML Libraries**: Scikit-learn, Pandas, NumPy
* **Model Deployment**: Joblib

---

## ⚙️ How It Works

1. User enters transaction details in the web interface
2. Data is sent to Flask backend
3. Backend preprocesses data (scaling, formatting)
4. ML model predicts fraud or not
5. Result is displayed on UI

---

## 📈 Results

* Achieved high accuracy using Random Forest
* Improved fraud detection using behavioral features
* Reduced false positives compared to baseline model
